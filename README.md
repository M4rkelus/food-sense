## Подготовка к заупску

1. Убедиться что стоит [node.js](https://nodejs.org/en/)
2. Выполнить команду
   ```bash
   npm install
   ```

## Работа с решением

Для единичной сборки решения нужно выполнить комманду:

```bash
npm run build:scss
```

Для запуска дев-сборки нужно выполнить команду:

```bash
npm run watch
```
